# CLAUDE.md

## 프로젝트 개요
- back-end, front-end 폴더 나눠서 사용
- back-end : spring framework + mybatis 프로젝트
- front-end : javascript + html 프로젝트

## Rules (코딩 규칙)
- back-end : spring framework + mybatis 사용
- front-end : javascript + html 사용
- back-end, front-end 위에 지정해준거 이외의 프레임워크 또는 언어 사용시 사용자에게 허락받기
- API 호출은 반드시 try-catch로 감쌀 것
- 주석은 한국어로 작성
